# AI Visibility Checker

A free static website MVP for checking whether AI search engines can understand
and recommend a product.

## Files

- `index.html` - landing page and checker tool
- `style.css` - visual design
- `app.js` - scoring and report generation
- `privacy.html` - privacy policy page

## Publish on GitHub Pages

1. Open the GitHub repository.
2. Click **Add file**.
3. Click **Upload files**.
4. Upload all files in this folder.
5. Click **Commit changes**.
6. Open **Settings**.
7. Click **Pages** in the left sidebar.
8. Under **Build and deployment**, select **Deploy from a branch**.
9. Branch: `main`.
10. Folder: `/ (root)`.
11. Click **Save**.

The site URL should become:

`https://aivisibilitytools.github.io/ai-visibility-checker/`
